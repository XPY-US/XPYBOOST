Junk Cleaner — What Gets Deleted?

The cleaner targets safe-to-delete Windows junk that builds up over time and wastes valuable drive space.

1

User Temp Folder (%TEMP%)

App installation leftovers, logs and cached files. 100% safe to delete — Windows recreates what it needs automatically.

2

Windows Temp (C:\Windows\Temp)

System-level temp files. Safe to clear when no critical operations are running.

3

Prefetch Files

Windows pre-loads apps here for faster launches. Safe to clear — it rebuilds automatically over time.

4

Windows Update Cache

Downloaded update files you no longer need after updates are installed. Can save 1–5GB easily.

Close all apps before running the cleaner. Open apps may lock some temp files — they'll be skipped automatically.

RAM Optimizer — How It Frees Memory

On low-end laptops with 4GB or less RAM, memory management is critical. Here's exactly what the optimizer does.

1

Clears Standby Memory

Windows keeps old app data in RAM "just in case." Clearing the standby list frees that RAM for active apps immediately.

2

Disables Memory Compression

Windows compresses RAM to fit more, but this uses CPU cycles. On weak CPUs, disabling this can make things noticeably snappier.

3

ProcessIdleTasks

Forces Windows to run idle maintenance tasks immediately so they don't interrupt you during gaming or work later.

After optimization, open Task Manager → Performance → Memory to see the improvement yourself!

Service Killer — What Gets Disabled?

Windows runs dozens of background services you don't need. Here's what we kill and exactly why.

1

SysMain (Superfetch)

Pre-loads apps into RAM. Helpful on fast SSDs but absolutely kills HDD performance and wastes RAM on old laptops.

2

WSearch (Windows Search Indexing)

Constantly indexes your drive in the background. Tanks HDD performance. Search still works — just not instantly.

3

DiagTrack (Telemetry)

Sends usage data to Microsoft constantly. Disabling saves a bit of bandwidth and CPU that's being used for nothing.

4

Print Spooler (Optional)

Only disable if you have no printer. You'll be asked before this is touched. Re-enable via services.msc if needed.

All disabled services can be re-enabled anytime via services.msc. Nothing is permanently broken.

Visual Mode — What Eye Candy Gets Killed?

Eye candy looks nice but obliterates performance on weak integrated graphics. Here's what visual mode disables.

1

Window Animations

Minimizing and maximizing animations are disabled. Windows will snap open and closed instantly instead of sliding.

2

Transparency (Acrylic/Blur)

The frosted glass effect on taskbar and Start Menu. Looks cool but burns GPU constantly even when idle.

3

Taskbar Animations

Icon bounce and taskbar fade effects disabled. Clicking apps feels much more immediate and responsive.

4

Drop Shadows

Icon and window shadows disabled. Saves constant GPU rendering work on every frame.

Log out and back in — or restart — to see all visual changes fully applied.

// FREQUENTLY ASKED QUESTIONS
Is this actually safe to run?
Yes. Every command is a standard Windows command you could run manually yourself. View the full source on GitHub. We disable services, clear temp files and change registry values — all documented Microsoft operations.
Why does Windows Defender flag the .bat file?
Windows SmartScreen flags unknown .bat files by default — not because they're dangerous, but because they haven't been Microsoft-certified. Click "More info" → "Run anyway". The source code is fully public to verify.
Can I undo the changes?
Yes — almost everything is reversible. Services re-enable via services.msc. Visual effects restore via System Properties → Advanced → Performance Settings. Power plan changes back in Power Options.
Does it work on Windows 10?
Yes! LaptopBoost works on both Windows 10 and Windows 11. Most commands are identical across both versions. A few visual tweaks may look slightly different but function the same.
How often should I run it?
Run the Junk Cleaner and RAM Optimizer weekly. Service and power plan changes are permanent until you undo them — set those once and forget. Run Network Boost whenever your internet feels sluggish.
Why not just use CCleaner?
CCleaner has had security incidents, installs its own background software and pushes a paid version constantly. LaptopBoost does the same core cleaning with zero bloat, zero cost. Plus we handle service disabling and power plans which CCleaner doesn't touch.
