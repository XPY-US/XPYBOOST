**XPYBOOST** is a free, open source Windows batch script that applies
proven performance optimizations to low-end laptops — the same tweaks
that IT professionals have been using for years, packaged in a clean
gaming-themed interface anyone can use.

No installer. No account. No bloat. No $30/year subscription.  
Just download, right-click, run as administrator and watch your laptop wake up.

**Built and maintained by [XPY-US](https://github.com/XPY-US)**

## 🎮 Features

| # | Tool | What It Does |
|---|------|-------------|
| `01` | **⚡ Full Auto Boost** | Runs all 8 optimizations automatically. Recommended for first-timers. |
| `02` | **🗑 Junk Cleaner** | Nukes temp files, prefetch, update cache. Frees multiple GBs instantly. |
| `03` | **💾 RAM Optimizer** | Clears standby memory list and kills memory compression overhead. |
| `04` | **⚙ Service Killer** | Disables Superfetch, Telemetry, Search Indexing and other parasites. |
| `05` | **⚡ High Performance** | Forces CPU and GPU to run at maximum power — no more throttling. |
| `06` | **🌐 Network Boost** | Flushes DNS, resets TCP stack, enables auto-tuning. Better ping. |
| `07` | **🎮 Gaming Mode** | Preset specifically for active gaming sessions. Disables Game DVR. |
| `08` | **🔋 Battery Saver** | Optimized settings for unplugged usage. Extends battery life. |
| `09` | **🎨 Visual Stripper** | Kills animations, transparency and Aero blur. Snappier on weak iGPUs. |
| `S` | **📊 Health Score** | Scans your system and gives a performance score from 0–100. |
| `I` | **💻 System Info** | Displays full hardware specs and OS information. |

🔧 How It Works
XPYBOOST runs standard Windows commands that are built into every
copy of Windows — nothing exotic, nothing dangerous. Here's exactly
what happens under the hood:

🗑 Junk Cleaner
batch

del /q /f /s %TEMP%\*.*                          :: User temp folder
del /q /f /s C:\Windows\Temp\*.*                 :: Windows temp
del /q /f /s C:\Windows\Prefetch\*.*             :: Prefetch cache
net stop wuauserv                                 :: Stop Windows Update
del /q /f /s C:\Windows\SoftwareDistribution\... :: Update download cache
net start wuauserv                                :: Restart Windows Update
💾 RAM Optimizer
batch

rundll32.exe advapi32.dll,ProcessIdleTasks        :: Clear standby list
powershell Disable-MMAgent -MemoryCompression     :: Stop RAM compression
⚙ Service Killer
batch

sc stop "SysMain"    && sc config "SysMain"    start= disabled
sc stop "WSearch"    && sc config "WSearch"    start= disabled
sc stop "DiagTrack"  && sc config "DiagTrack"  start= disabled
sc stop "WerSvc"     && sc config "WerSvc"     start= disabled
⚡ High Performance Mode
batch

powercfg -setactive 8c5e7fda-e8bf-4a96-9a85-a6e23a8c635c
powercfg -setacvalueindex SCHEME_CURRENT ... 100  :: Min CPU = 100%
🌐 Network Boost
batch

ipconfig /flushdns
netsh winsock reset
netsh int ip reset
netsh int tcp set global autotuninglevel=normal
netsh int tcp set global chimney=enabled
🎮 Gaming Mode
batch

reg add "...SystemProfile" /v SystemResponsiveness /d 0  :: Max game priority
reg add "...Tasks\Games" /v "GPU Priority" /d 8          :: GPU boost
reg add "...GameConfigStore" /v GameDVR_Enabled /d 0     :: Kill Game DVR
🎨 Visual Stripper
batch

reg add "...VisualEffects" /v VisualFXSetting /d 2       :: Best performance
reg add "...Personalize" /v EnableTransparency /d 0      :: No transparency
reg add "...Advanced" /v TaskbarAnimations /d 0           :: No animations

🛡️ Safety
Is XPYBOOST safe to run?
Yes. Every single command is a standard Windows operation.
You can read the entire script — it's less than 500 lines.

Concern	Reality
Malware?	Zero. Read every line at XPYBOOST.bat
Data collection?	None. The script has no network calls
Permanent changes?	Almost nothing. Services re-enable in services.msc
Antivirus flags?	SmartScreen flags ALL unsigned .bat files by default
Registry edits?	Only visual effect settings — fully reversible
How to Undo Everything
Change	How to Revert
Disabled services	Win + R → services.msc → find service → set to Automatic
Visual effects	Win + R → sysdm.cpl → Advanced → Performance → Settings
Power plan	Control Panel → Power Options → select Balanced
DNS changes	Network Settings → Change adapter options → IPv4 → Auto DNS
Registry edits	System Properties → Performance Options → Visual Effects → Let Windows choose

❓ FAQ
<details> <summary><b>Is XPYBOOST actually safe?</b></summary>
Yes. Every command is a standard Windows operation you could run
yourself in an admin cmd window. The full source is right here in
this repo — XPYBOOST.bat. Read it before running. It's under 500 lines.

</details><details> <summary><b>Why does Windows Defender / SmartScreen flag it?</b></summary>
Windows SmartScreen flags all unsigned .bat files by default,
regardless of content. It's a code-signing trust system, not malware
detection. Click More info → Run anyway. The source is fully
public here to audit.

</details><details> <summary><b>Can I undo all the changes?</b></summary>
Yes. Almost everything is reversible:

Services → services.msc
Visual effects → System Properties → Performance
Power plan → Control Panel → Power Options
Full rollback → Win + R → rstrui (System Restore)
The boost log on your Desktop lists exactly what was changed.

</details><details> <summary><b>Does it work on Windows 10?</b></summary>
Yes. XPYBOOST is tested on both Windows 10 and Windows 11.
All commands are compatible with both versions.

</details><details> <summary><b>How often should I run it?</b></summary>
Junk Cleaner + RAM Optimizer → Run weekly
Service Killer + Power Plan → Run once, it's permanent until reverted
Network Boost → Run whenever your connection feels slow
Health Score → Run anytime to check your PC's state
</details><details> <summary><b>Why not just use CCleaner?</b></summary>
CCleaner had a verified malware incident in 2017,
installs its own background startup service, constantly nags you to
upgrade and doesn't handle service disabling or power plan changes.

XPYBOOST does everything CCleaner does plus more — with zero bloat,
zero cost and you can read every single line before running.

</details><details> <summary><b>Will this void my warranty?</b></summary>
No. These are all standard Windows operations. Nothing at the
hardware or firmware level is touched.

</details><details> <summary><b>My antivirus deleted the file. What do I do?</b></summary>
Add an exclusion for XPYBOOST.bat in your antivirus settings,
or temporarily disable real-time protection before downloading.
The file contains no executable code — only Windows batch commands.

</details>
