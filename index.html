Getting Started — First Time Setup

Brand new to XPYBOOST? Get running in under 2 minutes with these steps.

1

Download the .bat file

Go to the Download page and grab the file. Save it anywhere — Desktop is easiest to find.

2

Right-click → Run as Administrator

This is mandatory. Without admin rights, service changes, power plans and registry edits won't apply.

3

Hit Option 1 — Full Auto Boost

For first-timers, press 1 and Enter. All 8 optimizations run automatically in sequence.

4

Restart your PC

Service disables and visual effect changes need a restart to fully apply. Always restart after running.

  Run the Junk Cleaner and RAM Optimizer weekly. Run the rest once and forget about it.

Junk Cleaner — What Gets Deleted

Targets safe Windows junk that builds up constantly and wastes drive space.

1

User Temp Folder (%TEMP%)

App install leftovers, logs, crash dumps. Totally safe to nuke — Windows recreates what it needs automatically.

2

Windows Temp (C:\Windows\Temp)

System-level temp files. Safe to wipe when nothing critical is running.

3

Prefetch Files

Windows pre-loads app data here. Safe to clear — it rebuilds over the next few days automatically.

4

Windows Update Cache

Old update packages you'll never need again. Can reclaim 1–5GB instantly.

  Close all apps first. Open processes may lock some temp files — they'll just be skipped.

RAM Optimizer — How Memory Gets Freed

On 4GB laptops every MB matters. Here's what the optimizer actually does to your memory.

1

Clears the Standby List

Windows keeps dead app data in RAM "just in case." Clearing it frees that memory instantly for whatever you're actually running.

2

Disables Memory Compression

Windows compresses RAM to squeeze more in, but this uses CPU cycles on every memory access. On weak CPUs this trade-off kills performance.

3

ProcessIdleTasks

Forces Windows background maintenance tasks to run immediately instead of interrupting you at the worst possible moment.

  After optimization, check Task Manager → Performance → Memory. You'll see the difference immediately.

Service Killer — What Gets Terminated

Windows runs 50+ background services by default. Most of them are useless on a low-end laptop.

1

SysMain (Superfetch)

Pre-loads "frequently used" apps into RAM. On an HDD this creates constant 100% disk usage. On 4GB RAM it wastes what little you have.

2

WSearch (Search Indexing)

Crawls your entire drive constantly in the background. Absolutely destroys HDD performance. Search still works — just not instant.

3

DiagTrack (Telemetry)

Sends your usage data to Microsoft every few minutes. Disabling it saves both bandwidth and the CPU cycles it steals.

4

Print Spooler (Optional)

You're asked before this one is touched. Only disable it if you genuinely don't have a printer connected.

  Everything is re-enableable via services.msc (Win + R → services.msc). Nothing is permanently broken.

Visual Stripper — What Eye Candy Gets Killed

Your iGPU is weak. Stop making it render useless animations 60 times per second.

1

Window Animations

Minimize and maximize animations are disabled. Windows snaps open and closed instantly — which is what you actually want.

2

Acrylic Transparency

The frosted glass blur on taskbar and Start Menu. Looks nice but the GPU re-renders it constantly even when nothing is happening.

3

Taskbar Animations

Icon bounce effects and taskbar fade transitions. Disabling them makes clicking apps feel dramatically more responsive.

4

Shadow Effects

Window and icon drop shadows. Every shadow is re-rendered every frame. On an iGPU this actually matters.

  Sign out and back in — or restart — to see all visual changes take full effect.

// FAQ
Is XPYBOOST actually safe?
Yes. Every single command is a standard Windows operation you could run yourself in cmd. View the full source on GitHub at github.com/XPY-US. We clear temp files, disable services and change registry values — all documented Microsoft operations.
Why does Defender flag the .bat file?
Windows SmartScreen flags ANY unsigned .bat file by default regardless of content. It's a trust system, not a malware detection. Click "More info" → "Run anyway." The source code is fully public to audit.
Can I undo everything?
Yes. Services re-enable in services.msc (Win+R → services.msc). Visual effects restore in System Properties → Advanced → Performance → Settings. Power plan resets in Control Panel → Power Options. The boost log on your Desktop shows exactly what was changed.
Windows 10 or 11 only?
Both work fine. All commands are compatible with Windows 10 and Windows 11. A few visual tweaks render slightly differently between versions but function identically.
How often should I run it?
Junk Cleaner and RAM Optimizer — run weekly. Service kills and power plan changes are permanent until you revert them — set once and done. Network Boost — run whenever your connection feels off.
Why not just use CCleaner?
CCleaner had a verified malware incident in 2017, installs its own startup service, nags you to upgrade constantly and doesn't handle service kills or power plans. XPYBOOST does everything CCleaner does plus more — with zero bloat and zero cost. And you can read every line before running.
